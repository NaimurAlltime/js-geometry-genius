Question-3: HWhat is event bubble?

Answer: Event bubbles are the concept of invoking event handler sequences where the event handlers are nested in other components. The sequence of events is that all its ancestors and children are nested along a path. All the children can be accessed through the parent using the event bubble.

Question-4: What is a callback function and why will you use it?

Answer: A callback function is a function that is passed as an argument to another function, which is then called inside the outer function. Callback functions are often executed in another task once an event occurs or a task completes. The advantage of callback functions is that they can be used repeatedly so that the code does not repeat.
